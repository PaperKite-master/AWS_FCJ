###  [Blog 1 - Simplifying multi-account storage with Amazon EFS and Amazon EKS](3.1-Blog1/)
This blog guides you through setting up cross-account mounting of Amazon EFS file systems on Amazon EKS clusters. Organizations increasingly adopt multi-account AWS strategies to enhance security, governance, and operational efficiency. With Amazon EFS, you can flexibly share POSIX-compliant files across multiple accounts while mounting the same file system to multiple Amazon EKS clusters. The article provides step-by-step instructions for configuring VPC peering, route tables, security groups, Route 53 DNS resolution, IAM roles, and the EFS CSI Driver to enable secure cross-account data sharing while maintaining account isolation.

###  [Blog 2 - Creative coding for everyone: Pi Jam expands digital skills for underserved learners in India with AWS](3.2-Blog2/)
This blog showcases how Pi Jam Foundation uses AWS to bridge the digital divide in India's education system. Despite India's National Education Policy 2020 emphasizing technology in education, many students in rural and public schools lack reliable connectivity and devices. Pi Jam built Code Mitra, a cloud-native platform that delivers creative coding education using block-based programming. The platform has reached 1.14 million students across 95% of India's districts, running entirely on AWS services including Lambda, Aurora, S3, CloudFront, API Gateway, ElastiCache, and Amazon Bedrock. The article highlights the platform's impact, integration with national learning management systems, support from AWS Education Equity Initiative, and future plans for AI-powered personalized learning.
